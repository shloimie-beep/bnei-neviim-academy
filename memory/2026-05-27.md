All services are live. Here is the research summary:

**TRANSCRIPTION APIs**

| Service | Best For | Cost | Notes |
|---|---|---|---|
| **OpenAI Whisper** | General purpose, Hebrew support | $0.006/min | Best accuracy for the price. Already have OpenAI key in repo. Supports diarization (speaker labels) via API. |
| **AssemblyAI** | Real-time + summarization | $0.37/hr | Has built-in features: auto chapters, sentiment, PII redaction. Good if you want "smart transcripts" out of the box. |
| **Deepgram Nova-2** | Speed, keyword boosting | $0.0043/min | Fastest. Good for batch processing large volumes. |
| **Rev.ai** | Accuracy purist | $0.02/min | Human-level accuracy, slower, more expensive. |

Recommendation: **OpenAI Whisper** - you already pay OpenAI, it handles Hebrew well, and the API is simple. If you need speaker labels or auto-summaries later, switch to AssemblyAI.

**VIDEO EDITING APIs**

| Service | What It Does | Cost | Notes |
|---|---|---|---|
| **Shotstack** | JSON-driven video assembly (clips, text, audio) | $0.05-0.15/render | Best fit for repurposing: take a video, cut segments, overlay text, render new versions. Pure API, no UI needed. |
| **HeyGen** | AI avatars + talking head generation | $2-3/min | Overkill unless you want AI presenters. |
| **Cloudinary** | Transformations, trimming, overlays | Free tier generous | Good for simple ops: trim, resize, add watermark. Not true editing. |
| **Mux** | Video hosting + lightweight processing | Usage-based | More for delivery than editing. |

Recommendation: **Shotstack** for the repurposing pipeline. You send a video + edit instructions (trim points, text overlays, output format) and it renders. Fits the "parse once, output many" model.
