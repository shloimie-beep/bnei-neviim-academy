Family Accountability

Six steps. ~15 minutes. You can do them in order on your phone.

1

Open your bot, send any message

Telegram link → Open @shlomofam_bot

Send anything like "hi". This registers your chat so the bot can ping you later.

2

Deploy the cloud accounts

Easiest: paste the agent prompt you have into your remote-Claude session and it sets up Supabase + GitHub + Railway + Resend for you. Otherwise, manually:

Supabase GitHub repo Railway project Resend

  • Supabase: create project family-accountability, run supabase-schema.sql, then supabase-migration-002.sql
  • Railway: deploy from the GitHub repo, paste env vars from .env.local
  • Note the deployed URL — you'll need it in step 3
3

Open the app on your phone, sign in

Replace <URL> with your Railway URL:

<URL>/parent/login

Enter your email → magic link arrives → tap it → onboarding wizard walks you through the rest (kid PINs, install on home screen, first family meeting, invite Ahuva).

4

Set the bot webhooks

One command on your PC:

node scripts/set-webhooks.mjs https://<URL>

Or paste each curl from scripts/set-webhooks.mjs manually if you'd rather see them run.

5

Forward Ahuva her invite

She does the same as you, on her phone:

  • Telegram @ahuvafam_bot → send "hi"
  • <URL>/parent/login → her email → magic link → onboarding
6

Print the kid QR codes

After deploy, open these in your browser and save the PNGs:

  • <URL>/api/qr/menachem
  • <URL>/api/qr/esther

Tape to each tablet. A scan opens the PIN entry directly.

What you and Ahuva can do from the bots

Type free-text questions in either bot — it goes to Claude with the family data only:

  • "How did Esther do this week?"
  • "What goals were rejected and why?"
  • "Add a task for Menachem to take out the trash" (manually use /addgoal for now)

Slash commands: /today, /streak, /meetings, /addgoal <kid> "title", /freeze <kid>, /unfreeze <kid>, /dashboard, /help.
