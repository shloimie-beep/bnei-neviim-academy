# Core Beliefs

Status: draft seed

These are durable BNA principles. Add only ideas that should guide many future pieces of communication, not one-time class details.

## Principles

- Torah learning should meet the whole child: mind, heart, middos, curiosity, confidence, and responsibility.
- Parent communication should be warm, honest, grounded, and practical.
- Student growth should be tracked through real moments: questions asked, goals chosen, private check-ins, and decisions made.
