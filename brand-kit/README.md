# BNA Brand Kit

This folder is the repo-side source of truth for BNA voice, beliefs, parent messaging, and teaching principles.

Google Drive mirrors this as `BNA V2 / BNA Brand Kit` so the working docs are easy to access from the phone and browser.

## Rules

- Do not dump full transcripts here.
- Add durable principles only after review.
- Store transcript-derived ideas as suggestions first.
- Avoid duplicates by checking whether the idea already exists in the relevant file.
- Keep platform-specific tone separate: WhatsApp, parent email, newsletter, blog, YouTube, and social posts should each get their own prompt/tone notes over time.
